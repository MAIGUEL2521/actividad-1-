Este proyecto consiste en el desarrollo de un sitio web informativo y funcional para la gestión de créditos financieros. El sistema está compuesto por tres páginas principales: un catálogo donde se presentan los diferentes tipos de créditos disponibles, un simulador que permite al usuario buscar y comparar opciones, y un formulario para solicitar un crédito de manera sencilla.
